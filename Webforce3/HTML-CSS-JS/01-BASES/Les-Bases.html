Les Balises DIV :

La Balise DIV, est une balise neutre de type block. Elle me servira à structurer ma page.

Elles agissent sur un Axe Vertical et donc génère un retour à la ligne.
Je pourrais toutefois changer leurs comportements avec du CSS.

Les Balises Inline elles, agissent sur un Axe Horizontal et donc ne génère aucun retour à la ligne.

Les Formulaires :

  • La Balise FORM permet de créer un Formulaire
  • name : Permet de donner un Nom au Formulaire
  • action : Cet attribut est OBLIGATOIRE et permet de renseigner le script, la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du Formulaire. Cette page va recevoir l'ensemble des données saisie par mon utilisateur.
  • method : Permet de préciser quelle méthode [GET|POST] doit être utilisée par le navigateur pour transmettre les données saisies vers la page action
